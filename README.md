# Diciembre Playlist Pro

Generador de playlists M3U para emisoras de radio.

## Funciones

- Carpetas por categorías
- Subcarpetas automáticas
- Secuencias personalizadas
- Música sin repetir hasta agotar carpeta
- IDs, Promos y Comerciales con repetición
- Exportación M3U / M3U8
- Historial de reproducción
